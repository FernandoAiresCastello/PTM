RECT
Fill a rectangle in the tilebuffer with copies of tile from tile register
Syntax:
RECT x1, y1, x2, y2
Fill a rectangle in the tilebuffer with copies of tile from tile register
  • x1 = First tilebuffer column
  • y1 = First tilebuffer row
  • x2 = Last tilebuffer column
  • y2 = Last tilebuffer row
