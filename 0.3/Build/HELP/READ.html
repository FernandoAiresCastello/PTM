READ
Read from data record file
Syntax:
READ var
Read data from currently open record file, store value in given variable, then moves the read pointer to the next record.

To read data, the file must be opened with the OPEN command in mode "R" (read), and the end-of-file must not be reached. To check for end-of-file, use the EOF command.
  • var = Variable identifier that will receive the value read
