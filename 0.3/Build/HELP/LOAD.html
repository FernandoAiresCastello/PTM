LOAD
Load file from disk
Syntax:
LOAD filename
Load the given file from disk. The file format is determined by the filename extension. See section below for supported file formats.
  • filename = The name of the file to load
Valid file formats:
The PTML interpreter only has support for loading the following file formats:
  • .PTM = PTML program
  • .PAL = Color palette (256 entries)
  • .CHR = Character set (256 entries)
  • .BUF = Tile buffer
  • .DAT = Data records (can't be loaded by the LOAD command; use the OPEN and READ commands instead)
