SAVE
Save file to disk
Syntax:
SAVE filename
Save code or data to given file. What exactly gets saved (and consequently the file format) is determined by the filename extension. See section below for supported file formats.
  • filename = The name of the file to save
Valid file formats:
The PTML interpreter only has support for saving the following file formats:
  • .PTM = PTML program. Will save the current program in memory
  • .PAL = Color palette (256 entries). Will save the current color palette in memory
  • .CHR = Character set (256 entries). Will save the current character set in memory
  • .BUF = Tile buffer. Will save the current tile buffer in memory
  • .DAT = Data records (can't be saved by the SAVE command; use the OPEN and WRITE commands instead)
