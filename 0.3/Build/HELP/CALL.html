CALL
Call subroutine that starts at specified label
Syntax:
CALL label
Unconditionally call subroutine
CALL.E a, b, label
Call subroutine if a = b
CALL.NE a, b, label
Call subroutine if a != b (not equal)
CALL.G a, b, label
Call subroutine if a > b
CALL.GE a, b, label
Call subroutine if a >= b
CALL.L a, b, label
Call subroutine if a < b
CALL.LE a, b, label
Call subroutine if a <= b
CALL.K key, label
Call subroutine if specified key is pressed
CALL.NK key, label
Call subroutine if specified key is not pressed
