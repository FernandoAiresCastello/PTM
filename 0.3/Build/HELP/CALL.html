CALL
Call subroutine that starts at specified label
Syntax:
CALL label
Unconditionally call subroutine
CALL.E a, b, label
Call subroutine if a = b
CALL.NE a, b, label
Call subroutine if a != b (not equal)
CALL.G a, b, label
Call subroutine if a > b
CALL.GE a, b, label
Call subroutine if a >= b
CALL.L a, b, label
Call subroutine if a < b
CALL.LE a, b, label
Call subroutine if a <= b
CALL.K scancode_name, label
Call subroutine if specified key is pressed. Use scancode names below
CALL.NK scancode_name, label
Call subroutine if specified key is not pressed. Use scancode names below
Scancode names to use with CALL.K and CALL.NK:
Scancode names are strings and as such should be used with double quotes.
  • RIGHT
  • LEFT
  • UP
  • DOWN
  • SPACE
  • RETURN
  • BACKSPACE
  • TAB
  • DELETE
  • INSERT
  • HOME
  • END
  • PAGEUP
  • PAGEDOWN
  • LSHIFT
  • RSHIFT
  • LCTRL
  • RCTRL
  • LALT
  • RALT
  • PAUSE
  • PRINTSREEN
  • ESCAPE
  • 0 through 9
  • A through Z
  • F1 through F12
