FILES
List user files on disk
Syntax:
FILES
List all files on disk, that are in the user root folder (USR)
FILES filter
List files on disk, that are in the user root folder (USR), that start or end with the given filter string. To filter by the last characters of the filename, start the filter string with an asterisk (*). Otherwise, the list will be filtered by the first characters of the filename.
  • filter = String used to filter the file list
