PRINTF
Print formatted text on screen at current cursor position
Syntax:
PRINTF format, value
Print formatted text on screen at current cursor position. The format string is similar to the C language. However, the only supported format specifiers are %i (integer) and %x (hexadecimal), with optional padding zeroes (e.g. %05i). Only a single specifier is interpreted in the string, because PTML does not support variable length arguments.
  • format = The format string (similar to the C language)
  • value = The value (or variable) to be formatted in the string
