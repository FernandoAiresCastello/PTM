CHAR
Get character at index of string
Syntax:
CHAR var, string, index
Get the character at specified index of string, then store it in the given variable. The character itself is stored, not its ASCII value. To get the ASCII value of the character, the ASC command must be used.
  • var = Variable identifier
  • string = The source string
  • index = The index in the source string from which to get the character
