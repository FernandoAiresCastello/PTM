COLOR
Change foreground/background colors of tiles and/or color of the screen border
Syntax:
COLOR fgc
Change foreground color of tiles
  • fgc = Palette index for the foreground color
COLOR fgc, bgc
Change foreground and background color of tiles
  • fgc = Palette index for the foreground color
  • bgc = Palette index for the background color
COLOR fgc, bgc, border
Change foreground and background color of tiles, and screen border color
  • fgc = Palette index for the foreground color
  • bgc = Palette index for the background color
  • border = Palette index for the screen border color
