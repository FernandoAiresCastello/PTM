RND
Get a random number
Syntax:
RND var, min, max
Generate a random number between min and max (inclusive), and store it in the given variable
  • var = Variable identifier
  • min = Minimum value
  • max = Maximum value
RND var, max
Generate a random number between 0 and max (inclusive), and store it in the given variable
  • var = Variable identifier
  • max = Maximum value
