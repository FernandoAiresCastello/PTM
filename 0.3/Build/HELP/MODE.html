MODE
Switch color mode
Syntax:
MODE mode
Switch the color mode.

Monochrome mode is the default, where every tile on the screen has the same foreground and background color, and the COLOR, INK and PAPER commands change the color of all tiles on the screen at once.

Multicolor mode prevents the COLOR, INK and PAPER commands to change existing tiles, and instead only sets the color of subsequent tiles.
  • mode = 0: monochrome; 1 = multicolor
