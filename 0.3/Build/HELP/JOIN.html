JOIN
Concatenate two strings
Syntax:
JOIN var, text_1, text_2
Concatenate text_1 with text_2, then store result in given variable
  • var = Variable that will receive the two concatenated texts
  • text_1 = First text
  • text_2 = Second text
JOIN var, text
Append text to the end of string stored in the given variable
  • var = Variable to which the text will get appended
  • text = Text to append
