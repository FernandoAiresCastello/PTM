GOTO
Transfer execution to code starting at specified label
Syntax:
GOTO label
Unconditionally go to label
GOTO.E a, b, label
Go to label if a = b
GOTO.NE a, b, label
Go to label if a != b (not equal)
GOTO.G a, b, label
Go to label if a > b
GOTO.GE a, b, label
Go to label if a >= b
GOTO.L a, b, label
Go to label if a < b
GOTO.LE a, b, label
Go to label if a <= b
GOTO.K key, label
Go to label if specified key is pressed
GOTO.NK key, label
Go to label if specified key is not pressed
