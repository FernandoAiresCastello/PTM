GOTO
Transfer execution to code starting at specified label
Syntax:
GOTO label
Unconditionally go to label
GOTO.E a, b, label
Go to label if a = b
GOTO.NE a, b, label
Go to label if a != b (not equal)
GOTO.G a, b, label
Go to label if a > b
GOTO.GE a, b, label
Go to label if a >= b
GOTO.L a, b, label
Go to label if a < b
GOTO.LE a, b, label
Go to label if a <= b
GOTO.K scancode_name, label
Go to label if specified key is pressed. Use scancode names below
GOTO.NK scancode_name, label
Go to label if specified key is not pressed. Use scancode names below
Scancode names to use with GOTO.K and GOTO.NK:
Scancode names are strings and as such should be used with double quotes.
  • RIGHT
  • LEFT
  • UP
  • DOWN
  • SPACE
  • RETURN
  • BACKSPACE
  • TAB
  • DELETE
  • INSERT
  • HOME
  • END
  • PAGEUP
  • PAGEDOWN
  • LSHIFT
  • RSHIFT
  • LCTRL
  • RCTRL
  • LALT
  • RALT
  • PAUSE
  • PRINTSREEN
  • ESCAPE
  • 0 through 9
  • A through Z
  • F1 through F12
