LIST
List current program
Syntax:
LIST
List all lines of the program currently stored in memory
LIST label
List lines of the program currently stored in memory, starting from the given label
  • label = The program label
LIST line_number
Print the given line of the program currently stored in memory
  • line_number = The number of the line to print
LIST first_line, last_line
List lines of the program currently stored in memory, between the given first and last lines
  • first_line = The number of the first line
  • last_line = The number of the last line
