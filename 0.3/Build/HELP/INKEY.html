INKEY
Get code of last pressed key
Syntax:
INKEY var
Store code of last pressed key in the given variable.

This command is subject to the keyboard repeat rate and delay. For continuous input, check the scancode of a currently pressed key and then branch to a subroutine, using any of the GOTO.K, GOTO.NK, CALL.K, CALL.NK commands.

See the next section for common keycodes.
  • var = Variable identifier
