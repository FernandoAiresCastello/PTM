SUBS
Get substring
Syntax:
SUBS var, source_string, first_index, last_index
Extract a substring from the given string, that starts at the first_index and ends at the last_index. The source string is not modified.
  • var = Variable identifier that will receive the substring
  • source_string = Source string
  • first_index = The first index of the substring
  • last_index = The last index of the substring
SUBS var, source_string, first_index
The same as the previous syntax, but the last_index is implied to be the index of the last character of the source string.
  • var = Variable identifier that will receive the substring
  • source_string = Source string
  • first_index = The first index of the substring
